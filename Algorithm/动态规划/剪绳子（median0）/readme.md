**思路二**：数论，贪心思想

任何大于1的数都可由2和3相加组成（根据奇偶证明）

因为2*2=1*4，2*3>1*5, 所以将数字拆成2和3，能得到的积最大

因为2*2*2<3*3, 所以3越多积越大 时间复杂度O(n/3)，用幂函数可以达到O(log(n/3)), 因为n不大，所以提升意义不大，我就没用。 空间复杂度常数复杂度O(1)
